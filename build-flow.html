React App Build Flow

Overview of how this Vite + React project is built and how the runtime app is composed.

1. Build entry

Command: npm run build

This executes vite build from package.json.

Build config: vite.config.js

Uses the React plugin and Babel React Compiler preset:

react(), babel({ presets: [reactCompilerPreset()] })
Vite reads index.html as the application entry point and then bundles the imported module graph.

2. Build graph

index.html

HTML shell containing <div id="root"> and module script:

<script type="module" src="/src/main.jsx">

↓

src/main.jsx

Root React entry. Imports styles and renders the app tree.

createRoot(...).render(...)

↓

src/App.jsx

Main shell containing navigation and route outlet.

<AppNavBar />
<AppRoutes />

↓

src/routes/AppRoutes.jsx

Route definitions for /login, /dashboard, /, and fallback *.

3. Runtime component tree

index.html
  └─ src/main.jsx
        └─ <StrictMode>
              └─ <Provider store={store}>
                    └─ <BrowserRouter>
                          └─ <ThemeProvider>
                                └─ <App />
                                      ├─ <AppNavBar />
                                      └─ <main class="page-shell">
                                            └─ <AppRoutes />
                                                  ├─ /login      → <LoginPage />
                                                  ├─ /dashboard  → <DashboardPage />
                                                  ├─ /            → <DashboardPage />
                                                  └─ *            → <DashboardPage />
State providers: Redux store is available across the app. The theme provider updates document.documentElement.dataset.theme from Zustand state.

4. Key files

  • index.html – application HTML shell
  • src/main.jsx – React root and providers
  • src/App.jsx – app layout and route container
  • src/routes/AppRoutes.jsx – router setup
  • vite.config.js – Vite plugins and build config
  • package.json – build script and dependency list

5. Output

Production build emits optimized files into the dist/ folder.

Vite bundles JavaScript, CSS, and static assets, performing tree-shaking and minification.
